संदर्भ

यह परिशिष्ट एक पृष्ठ पर करेल प्रोग्रामिंग भाषा की संरचना को परिभाषित करता है।

बेस कारेल कमैंट्स:

move();
turnLeft();
putBeeper();
pickBeeper();

कारेल कार्यक्रम संरचनाएं:

/* टिप्पणियां किसी भी हिस्से में शामिल की जा सकती हैं
 * का a कार्यक्रम. वे शुरु साथ_में a स्लेश-तारा
 * and end with a star-slash.
 */
 public class नाम extends सुपरकारेल {

   public void run() {
       कोड निष्पादित करने के लिए
   }

    निजी तरीकों की घोषणा
}

}
