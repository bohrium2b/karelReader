Referans

Bu ek, Karel programlama dilinin yapısını tek bir sayfada tanımlar.

Karel üssü emirleri:

move();
turnLeft();
putBeeper();
pickBeeper();

Karel program yapıları:

/* Yorumlar herhangi bir bölüme dahil edilebilir
 * arasında a program. Onlar başla ile a yırtmaç-star
 * and end with a star-slash.
 */
 public class isim extends SüperKarel {

   public void run() {
       çalıştırılacak kod
   }

    özel yöntemlerin beyanları
}
